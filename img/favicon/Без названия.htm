Описание курса

Frontend разработчик верстает сайты и шаблоны для различных CMS, разрабатывает пользовательский интерфейс и внедряет в него скрипты отвечающие за анимацию и логику на веб-страницах. Обычно Frontend разработчик работает в команде с дизайнером и Backend разработчиком.

На нашем курсе ты научишься строить красивые и интерактивные сайты используя фундаментальные знания HTML, CSS и JavaScript – трех главных языков веб-разработки на которых построены практически все сайты. Это очень полезный набор навыков, так как эти языки использует почти каждая компания в мире, которой нужен веб-сайт или приложение для обращения к своим клиентам. Вот компании, которые используют эти технологии – Google, Facebook, Yahoo, eBay, Amazon, Yandex, Beeline и UMS.

  • 3 дня в неделю по 90 минут

  • 600 000 сум/мес

    • Что ты получаешь

    1. Умение верстать сайты используя HTML5 и CSS3
    2. Понимание JavaScript и умение создавать интерактивные страницы
    3. Опыт написания AJAX запросов
    4. Умение работать с GIT
    5. Навыки работы с Gulp и Sass
    6. Портфолио с проектами
    7. Консультация по успешному поиску работы

    HTML & CSS. I

    Базовый

    Основные принципы верстки

    1 месяц
    1. Вступление – как работает интернет, взаимодействие браузера и веб-сервера, основы HTTP.
    2. Знакомство с HTML – синтаксис, структура докумнта, основные теги.
    3. Основы CSS – синтаксис, базовые свойства
    4. CSS Box Model – основополагающий принцип в построении структуры HTML документа.
    5. Работа с текстом в CSS – форматирование, базовые свойства. Подключаем normalize.css.
    6. Позиционирование элементов в CSS.
    7. CSS background – установка фона для элементов различными способами.
    8. HTML Forms – форматирование элементов формы.
    9. Основы Photoshop для верстальщика.
    10. Нарезка шаблона PSD в HTML. Часть первая.
    11. Нарезка шаблона PSD в HTML. Часть вторая.
    12. Адаптация верстки под мобильные устройства.

    HTML & CSS. II

    Базовый

    Продвинутые способы верстки, методологии, автоматизация

    1 месяц
    1. Bootstrap – основы сетки, Fixed layout, Fluid layout, Responsive Layout, типографика, таблицы, списки.
    2. Bootstrap – элементы формы, навигация, кнопки, группы кнопок, изображения, иконки.
    3. Bootstrap – Panels, Bradcrumbs, Pagination, Labels & Badges, Progress Bars, Jumbotorn, Walls, Helper Classes
    4. CSS Transform, Transitions – Трансформации элементов (поворот, масштаб, сдвиг) с помощью CSS. Анимация переходов между состояниями элемента.
    5. CSS Animations – продвинутая анимация с помощью CSS.
    6. SVG Graphics – использование векторной графики в веб-приложениях. Преимущества, ограничения.
    7. CSS Flexbox – новая технология для верстки гибкой адаптивной разметки.
    8. HTML5 Audio/Video – использование видео и аудио в HTML коде.
    9. SASS – знакомство с препроцессором CSS. Преимущества, возможности.
    10. SASS – объявление переменных, встроенные функции, организация стилей разбиением на несколько файлов.
    11. SASS – миксины, медиа-выражения, управляющие конструкции.
    12. Методология BEM – подход к написанию стилей для крупных проектов.

    JavaScript. I

    Базовый

    Базовые представления о языке, синтаксисе, возможностях

    1 месяц
    1. Основы JS – краткая история, обзор возможностей, первый скрипт на JS.
    2. Типы данных в JS и принципы работы с ними.
    3. Основные операторы – Операторы и операнды. Приоритет операций. Операции сравнения. Побитовые операции.
    4. Управляющие конструкции – циклы, условный оператор if/else, оператор множественного выбора switch, тернарный оператор.
    5. Функции – Области видимости. Замыкания. Функциональные выражения. Именованные функциональные выражения. Рекурсия.
    6. Введение в объектно-ориентированное программирование (ООП), реализация в JS. Создание объектов в JS, литеральная форма, конструкторы, методы и свойства.
    7. Встроенные объекты JS – объекты Object, Math, Date, Array, Function.
    8. Работа со строками – поиск, замена, разбиение, конкатенация, преобразование встроенных объектов в строковое выражение.
    9. Объекты. Контекст вызова – Ключевое слово this. Преобразование объектов, создание объектов. Статические методы. Явное указание this (call, apply, bind).
    10. Работа с массивами в JS – способы обработки и преобразования данных.
    11. DOM. Документ и объекты страницы – дерево DOM. Работа с DOM из консоли. Навигация по DOM. Свойства узлов, атрибуты, стили, размеры, координаты и прочее. Добавление/удаление узлов.
    12. DOM. События – Введение в браузерные события, порядок их обработки. Объект события. Всплытие и перехват. Делегирование событий. Генерация событий. Действия браузера по умолчанию.

    JavaScript. II

    Базовый

    ООП, Ajax, регулярные выражения, обзор основных библиотек, тестирование

    1 месяц
    1. ООП в JavaScript – наследование и особенности реализации в JS, прототипное наследование – плюсы и минусы данного подхода.
    2. AJAX – Асинхронные запросы с помощью XMLHttpRequest, типы запросов, форматы JSON и XML.
    3. Регулярные выражения в JavaScript – паттерны и флаги, Regexp и String, квантификаторы, наборы и диапазоны.
    4. Продвинутая графика на основе Canvas + JS.
    5. Введение в jQuery – селекторы, взаимодействие с DOM, события jQuery.
    6. AJAX в jQuery, работа со стилями и анимация.
    7. jQuery UI и другие расширения jQuery. Пример использования.
    8. Разработка плагина jQuery.
    9. ES6 – новый стандарт языка. Обзор нововведений, применение на практике.
    10. Библиотеки для JS-разработчика. Микро-библиотеки, шаблонизаторы – список наиболее популярных и эффективных решений.
    11. Менеджеры пакетов для JS, утилиты для сборки и автоматизации на примере gulp.
    12. Тестирование: куда дальше? – Типы тестирования, тестирование в JavaScript, Jasmine.js, дальнейшее развитие.

    AngularJS. Основы

    Базовый

    MVC-фреймворк AngularJS. Основные понятия, принципы построения SPA приложений.

    1 месяц
    1. Представление о MVC-архитектуре, принципы работы SPA-приложений (Single Page Application)
    2. Знакомство с AngularJS, установка, основные понятия.
    3. Структура проекта, модули приложения в AngularJS.
    4. Привязка данных, подключение обработчиков событий.
    5. Обзор встроенных директив AngularJS.
    6. Работа с контроллерами, внедрение зависимостей.
    7. Сервисы, фабрики AngularJS. Основные понятия.
    8. $http - встроенные сервис AngularJS. Взаимодействие с бэкендом.
    9. Обзор REST-архитектуры, ее применение в AngularJS.
    10. Аутентификация в SPA-приложении.
    11. Создание кастомных директив AngularJS.
    12. Роутинг в SPA-приложении. Реализация в AngularJS на основе модуля ui-router.
